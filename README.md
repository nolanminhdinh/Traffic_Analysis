# Traffic_Analysis
Nghiên cứu ứng dụng dữ liệu không gian(vị trí) đa thời gian. Dự án tập trung vào phân tích ảnh hưởng của đèn giao thông đến với tình trạng tắc đường. 

Chạy python app.py ở file Flask trong cmd/powershell trước(không cần mở link localhost)

Xong chạy flutter run -d emulator-5554 trong cmd/powershell nếu muốn mở bằng điện thoại ảo ở file gps_collector_app 

Hoặc flutter run để mở bằng web chrome hoặc edge
